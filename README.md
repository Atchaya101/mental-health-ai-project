# Mental Health Support Chatbot

## 📌 Project Overview
This project is a Mental Health Support Chatbot developed using Python, Flask, and Streamlit. It provides basic emotional support by analyzing user input and generating appropriate responses.

## 🎯 Objective
To develop an AI-based chatbot that helps users express emotions and receive supportive responses, aligned with SDG Goal 3 (Good Health and Well-being).

## ⚙️ Technologies Used
- Python
- Flask (API)
- Streamlit (UI)

## 🧠 Features
- Sentiment detection
- AI-based response generation
- REST API integration
- Interactive user interface
